Web Resources

Resources on the web! One of the greatest benefits of the internet is the easy access to information, and very often, that information is free! Regardless of what profession you're in, you can bet you can find a resource for your interests! For an Application Developer, I narrowed down my most favorite resources to this list of 5! If you like these resources, consider adding them to your toolkit.

Adobe Color Picker

Rating:

Visit Site

Adobe Color Picker has been a go-to resource for myself for as long as I can remember, it's a simple and intuitive tool that allows you to easily design a color scheme and gives you access to the color mode of your choice. This tool can be useful for any developer that's using color in their applications, web developers would get plenty of use out of this though.

Bootstrap 4 Reference

Rating:

Visit Site

Keeping documentation for languages, frameworks, and really anything related to what you're developing is always a good practice. Often times, the source documentation will be able to give you the most straight forward answer as quick as possible. A reference for Bootstrap will be most useful to web developers since it's one of the most popular frameworks for building responsive, mobile-first sites.

Microsoft Docs

Rating:

Visit Site

Microsoft Docs is quite a broad source of information, however, it's been one of my most-used resources of all time. Everything is documented excellently, it has everything from C++, SQL Server, to .NET. Since so many different topics are covered in the MS Docs, it can be useful for a large range of developers. If you're using something from the Microsoft stack, know that you can find the answers here!

Stack Overflow

Rating:

Visit Site

Stack Overflow is an amazing collaboration platform, it has sections of its site dedicated to all sorts of diciplines in programming, application development, web development, gaming, you name it. If sifting through documentation can't answer your questions, the army of developers who frequent this site will definitely be able to help you out!

Packt Publishing

Rating:

Visit Site

Packt Publishing is a book/e-book publisher, and sadly, this resource is not free. What you can expect to find here are a pletora of books regarding many diciplines in software engineering and the best part of all, purchasing either the books or e-books will also gain you access to the sample source code for each book. With a little reading and working the sample projects of your interest, a lot can be learned here!
